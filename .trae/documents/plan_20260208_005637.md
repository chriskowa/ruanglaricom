# Rencana Modernisasi Race Master & Fitur Poster Peserta

Saya akan melakukan perombakan total pada UI Race Master agar terlihat modern (app-like experience), menambahkan Dark Mode, dan mengubah logika generator poster menjadi per peserta.

## 1. Backend: Generator Poster Individual
Membuat endpoint baru atau memodifikasi yang ada untuk menghasilkan poster spesifik per peserta.
- **Endpoint:** `POST /api/tools/race-master/sessions/{session}/poster/{participant_id}`
- **Desain Poster:**
  - Background: Custom upload atau default gradient modern.
  - Konten: Nama Peserta (Besar), BIB, Finish Time, Rank (Misal: "Finisher #1"), dan Logo Race.
  - Format: Portrait 1080x1920 (IG Story ready).
- **Logic:** Mengambil data spesifik peserta dari `RaceSessionLap` dan `RaceSessionParticipant`.

## 2. Frontend: UI/UX Redesign (Modern & Dark Mode)
Mengubah `race-master.blade.php` dengan desain system baru berbasis Tailwind CSS.

### A. Dark Mode Engine
- State `isDarkMode` di Vue (persisted di LocalStorage).
- Wrapper class `dark` pada root element.
- Palet warna:
  - **Light:** Slate-50 background, White cards, Indigo primary.
  - **Dark:** Slate-900 background, Slate-800 cards, Indigo-400 primary, Text Gray-100.

### B. Layout Modern
- **Header:** Sticky, Glassmorphism (blur effect), Toggle Dark/Light mode.
- **Timer (Race Screen):** Font Monospace besar, tampilan kontras tinggi agar terbaca di outdoor/sinar matahari.
- **Cards:** Rounded-2xl, subtle shadows, border tipis untuk dark mode contrast.
- **Animations:**
  - `TransitionGroup` untuk list peserta (animasi halus saat posisi klasemen berubah).
  - Efek "Flash" visual saat scan QR berhasil.

## 3. Fitur: Tombol Poster di Tabel Hasil
- Memindahkan tombol "Poster" dari header global ke dalam tabel hasil (`<tr>`).
- **Kolom Aksi:** Menambahkan tombol icon "IG Story" di sebelah tombol "Detail/Edit".
- **Flow:**
  1. Klik icon IG Story di baris peserta.
  2. Muncul Modal Preview (Loading state saat generate).
  3. Tampil gambar hasil generate.
  4. Tombol: "Download Image" & "Share" (Web Share API).

## 4. Improvements Tambahan
- **Sound Effects:** Menambahkan opsi suara "Beep" saat lap terekam (feedback audio penting untuk operator race).
- **Confetti:** Efek visual saat peserta finish.
- **Responsive Table:** Card view untuk mobile, Table view untuk desktop.
