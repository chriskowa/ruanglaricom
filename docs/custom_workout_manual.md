# Custom Workout Feature - User Manual

## Overview
The Custom Workout feature allows coaches to create their own workout templates that can be reused across different training programs. Coaches can choose to keep these workouts private or share them with other coaches.

## How to Access
1. Log in as a Coach.
2. Navigate to **Program Management**.
3. Click **Create New Program** or **Edit** an existing program.
4. In the **Workout Schedule** section, click the **Add Workout** button.
5. Select the **Custom Workout** tab in the modal.

## Creating a Custom Workout
1. Click **Create New Custom Workout**.
2. Fill in the form:
   - **Title**: Name of the workout (e.g., "Tempo Run 5k").
   - **Type**: Select category (Easy Run, Long Run, Tempo, Interval, Strength, Rest).
   - **Description**: Add instructions or details.
   - **Default Distance/Duration**: Set default targets.
   - **Thumbnail**: Upload an image (optional).
   - **Public Visibility**: Toggle "Make Public" if you want other coaches to use this workout.
3. Click **Save**.
4. The workout will appear in your custom list and can be dragged into the schedule.

## Editing/Deleting
- To **Edit**: Click the pencil icon on the workout card in the list. You can only edit workouts you created.
- To **Delete**: Click the trash icon. **Warning**: This action cannot be undone.

## Visibility Rules
- **Private Workouts**: Only visible to you.
- **Public Workouts**: Visible to all coaches in the system. Use this for generic templates.
- **System Workouts**: Standard workouts provided by the platform (read-only).
